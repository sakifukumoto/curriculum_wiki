# Ph.1 Java基礎

## ゴール

このフェーズが終わった時点で、以下ができる状態を目指します。

- クラス・継承・interfaceを使ったプログラムを自分で設計・実装できる
- 例外処理・Stream API・Optionalを適切に使える
- JUnit5でテストコードが書ける
- JDBCを使ってDBとやり取りできる

## 期間の目安

**2〜3ヶ月**（週20時間 × 8〜12週）

## 進め方

各課題は以下の3段階で構成されています。**教材を読んでから課題に進む必要はありません。**

```
Step 1: まず動かす  ← コードが用意されているのでそのまま動かす
Step 2: 理解する   ← 「なぜ動くか」を確認する設問に答える
Step 3: 自分で実装 ← 仕様だけ渡されるので自分で実装する
```

詰まったときだけ参考リンクや教材を参照してください。
