# Ph.1 Java基礎

## ゴール

- クラス・継承・interfaceを使ったプログラムを自分で設計・実装できる
- 例外処理・Stream API・Optionalを適切に使える
- JUnit5でテストコードが書ける
- JDBCを使ってDBとやり取りできる
- **バグの原因をエラーログから特定して修正できる**

## 期間の目安

**2〜3ヶ月**（週20時間 × 8〜12週）

## 進め方

各課題は「Step 1：動かす → Step 2：理解する → Step 3：実装する」の3段階で進めます。
**教材を全部読んでから課題に進む必要はありません。**

## フェーズ通過基準

- interfaceと抽象クラスの使い分け
- NullPointerExceptionが起きる原因と対処法
- Streamのmap・filter・reduceの違い
- トランザクションがなぜ必要か
- **エラーメッセージを見て、どこで何が起きているか説明できる**
