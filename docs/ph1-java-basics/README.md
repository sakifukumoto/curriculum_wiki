# Ph.1 Java基礎

## ゴール

このフェーズが終わった時点で、以下ができる状態を目指します。

- クラス・継承・interfaceを使ったプログラムを自分で設計・実装できる
- 例外処理・Stream API・Optionalを適切に使える
- JUnit5でテストコードが書ける
- JDBCを使ってDBとやり取りできる

## 期間の目安

**2〜3ヶ月**（週20時間 × 8〜12週）

## 進め方（重要）

### ステップ1：文法を学ぶ（インプット）
### ステップ2：写経する（手を動かす）
### ステップ3：課題に取り組む（アウトプット）

この3ステップを繰り返します。**いきなり課題に取り組まないでください。**
文法を理解してから課題に進むと、詰まる時間が大幅に減ります。
