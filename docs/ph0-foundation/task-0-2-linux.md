# 課題0-2：Linux基本コマンド

## この課題のゴール

ターミナルを使った基本的なファイル操作ができるようになる。

## 目安時間

5〜8時間

## 覚えるコマンド一覧

| コマンド | 意味 | 使い方の例 |
|---------|------|----------|
| `pwd` | 今いるディレクトリを表示 | `pwd` |
| `ls` | ファイル一覧を表示 | `ls -la` |
| `cd` | ディレクトリを移動 | `cd Documents` |
| `mkdir` | ディレクトリを作成 | `mkdir my-project` |
| `touch` | ファイルを作成 | `touch hello.txt` |
| `cat` | ファイルの中身を表示 | `cat hello.txt` |
| `rm` | ファイルを削除 | `rm hello.txt` |
| `cp` | ファイルをコピー | `cp a.txt b.txt` |
| `mv` | ファイルを移動・リネーム | `mv a.txt b.txt` |

> ⚠️ `rm` は元に戻せません。

## 課題

以下の操作をターミナルで実行し、スクリーンショットを撮って提出してください。

1. ホームディレクトリに `practice` ディレクトリを作成する
2. `practice` の中に `hello.txt` を作成し、自己紹介文を書く
3. `hello.txt` を `profile.txt` にリネームする
4. `practice` ディレクトリの中身を `ls -la` で表示する
