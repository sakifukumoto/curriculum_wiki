# 課題0-1：開発環境セットアップ

## この課題のゴール

以降の課題を進めるための開発環境を整える。**手順書通りに進めれば完了できます。**

## 目安時間

3〜5時間

## 手順

### 1. Javaのインストール

```bash
# Macの場合
brew install openjdk@21
java -version
# openjdk version "21.x.x" と表示されればOK
```

### 2. IntelliJ IDEA Community Editionのインストール

[公式サイト](https://www.jetbrains.com/idea/download/)からCommunity版をダウンロード。

### 3. GitHubアカウントの作成

[GitHub](https://github.com/)でアカウントを作成し、講師に自分のアカウント名を伝える。

### 4. リポジトリのclone

```bash
git clone https://github.com/sakifukumoto/curriculum_wiki.git
cd curriculum_wiki
```

## OK基準

- [ ] `java -version` でJava 21が表示される
- [ ] IntelliJ IDEAが起動する
- [ ] GitHubアカウントを作成し、講師に共有済み
- [ ] リポジトリをcloneできた
