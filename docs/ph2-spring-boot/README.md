# Ph.2 Spring Boot基礎

## ゴール

- Spring BootでREST APIを一から設計・実装できる
- DIの仕組みを理解し、適切なレイヤー分離ができる
- JPA/Hibernateを使ったDB操作ができる（N+1問題を理解・対処できる）
- Spring Securityで基本的な認証ができる

## 期間の目安

**3ヶ月**（週20時間 × 12週 = 240時間）
