# 課題4-3：AWSデプロイ

## この課題のゴール

AWSの基本サービスを使って本番相当の環境にアプリをデプロイできるようになる。

## 目安時間

15〜20時間

## 使用するAWSサービス

| サービス | 役割 |
|---------|------|
| EC2 | アプリサーバー（Spring Bootを動かす） |
| RDS | DBサーバー（MySQL） |
| S3 | 静的ファイルの保存（任意） |

## 仕様

課題4-1のDockerアプリをAWSにデプロイしてください。

### 構成

```
インターネット → EC2（Spring Boot on Docker） → RDS（MySQL）
```

### 手順

1. RDSでMySQLインスタンスを作成する
2. EC2インスタンスを作成し、Dockerをインストールする
3. EC2上でDockerコンテナを起動する
4. セキュリティグループを設定し、外部からAPIにアクセスできるようにする

## OK基準

- [ ] EC2のパブリックIPでAPIにアクセスできる
- [ ] DBがRDSに接続されている
- [ ] セキュリティグループが適切に設定されている（不要なポートを開けない）
- [ ] PR本文に「EC2 / RDS / セキュリティグループの役割説明」が書かれている
