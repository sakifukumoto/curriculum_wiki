# 📚 エンジニア育成カリキュラム

Java / Spring Boot 現場2〜3年相当を目指す育成カリキュラムです。

## 対象者

- プログラミング未経験者
- 週20時間程度の学習時間を確保できる方

## カリキュラム構成

| フェーズ | 内容 | 期間 |
|--------|------|------|
| [Ph.0 土台づくり](docs/ph0-foundation/README.md) | PC基礎・Linux・Git・SQL | 1ヶ月 |
| [Ph.1 Java基礎](docs/ph1-java-basics/README.md) | OOP・Stream・テスト・JDBC | 2〜3ヶ月 |
| [Ph.2 Spring Boot基礎](docs/ph2-spring-boot/README.md) | DI・REST API・JPA・Security | 3ヶ月 |
| [Ph.3 設計・品質・チーム開発](docs/ph3-design-team/README.md) | DB設計・テスト・Git Flow | 2〜3ヶ月 |
| [Ph.4 実践・インフラ](docs/ph4-infra/README.md) | Docker・CI/CD・AWS | 1ヶ月 |
| [Ph.5 実務模擬プロジェクト](docs/ph5-project/README.md) | 要件定義〜デプロイまで通し体験 | 2ヶ月 |

## 学習の進め方

1. 各フェーズの `README.md` を読み、ゴールを確認する
2. 課題ページの **仕様** を読んで実装する
3. **OK基準** を全て満たしたら講師に提出する
4. 講師のFBを受け、OKが出たら次の課題へ進む

## 困ったときは

1. まずエラーメッセージを読む（[エラーの読み方ガイド](docs/ph0-foundation/how-to-debug.md)）
2. Google・公式ドキュメントで調べる
3. 30分調べてもわからなければ講師に質問する
